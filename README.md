# BoostPanel

Nigeria's Premium SMM Reseller Platform

## Files

| File | Description |
|------|-------------|
| `landing-page.jsx` | Public homepage — hero, features, pricing, FAQ, auth modals |
| `smm-panel.jsx` | User dashboard — orders, wallet, services, referrals, support |
| `admin-login.jsx` | Admin authentication gate — hidden at /admin |
| `admin-panel.jsx` | Admin dashboard — orders, users, services, API, payments, tickets |

## Tech Stack
- **Frontend:** React (JSX artifacts)
- **Fonts:** Cormorant Garamond + Outfit + JetBrains Mono
- **Theme:** Luxury rose accent (#c47d8e), auto day/night toggle
- **Payments:** Paystack (NGN)
- **API Supplier:** MoreThanPanel
- **Currency:** Nigerian Naira (₦)

## Design System
- Day mode: warm cream (#f4f1ed) background
- Night mode: deep navy (#080b14) background  
- Auto-shifts based on local time (Day 7am–6pm, Night 7pm–6am)
- Frosted glass cards with backdrop blur
- Responsive: desktop, tablet, mobile breakpoints

## Status
UI build phase — all 4 pages complete. Backend integration pending.
