Per-platform connection model

  • Each platform connects independently. Connecting Instagram doesn't connect TikTok. The user manages each platform's session separately on their own device.
  • Tabs always show all 3 platforms, even unconnected ones â surfaces capability, signals "this works on multiple platforms."
  • Connected platforms show their suggested-cleanup count on the tab. Tapping switches the audit view to that platform.
  • Disconnected platforms show "Connect" instead of a count, slightly muted. Tapping swaps the page body for an inline connect prompt scoped to that platform's branding.
  • Inline connect stays on the same page â no modal, no full-page redirect. Single CTA, then the audit auto-runs after connection completes.

  • Below the page subtitle on connected platforms: "Last audited 2d ago Â· Re-scan" â small accent link, runs a fresh audit. Important because the user's following list changes daily.

SVG icon system

  • Single SVG sprite at the top of the document with all icons defined as <symbol>. Used via <use href="#i-..."/> throughout. Stroke-based, 1.8 stroke-width default for harmony, all use currentColor so theme switching works automatically.
  • No emoji anywhere. Every icon is a proper SVG with consistent visual weight.
  • Platform tab icons (IG, TikTok, X) are inline SVG with brand-correct colors and gradients.
